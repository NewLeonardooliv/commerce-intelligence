# Commerce Intelligence

Intelligent data analysis service powered by AI agents.

## Tech Stack

- **Runtime**: Bun
- **Framework**: Elysia.js
- **Language**: TypeScript
- **Architecture**: Modular feature-based

## Getting Started

### Prerequisites

- Bun >= 1.0.0

### Installation

```bash
bun install
```

### Configuration

```bash
cp .env.example .env
```

Edit `.env` with your configuration.

### Development

```bash
bun dev
```

### Production

```bash
bun start
```

## API Documentation

API documentation is available at `/swagger` when running in development mode.
